# A1. Discover security concepts used on campus

## Security Concepts Used on Campus

## 1) UWA Student Card
The UWA Student Card provides secure access to campus buildings like the labs and the library (after hours access). It uses electronic authentication to ensure only authorised students can enter and logs audits each entry and exit

## 2) Campus Security
UWA Security patrols the campus 24/7, responds to incidents and provides safety escorts. They maintain a secure environment for students, staff, and visitors.

## 3) Microsoft MFA Authenticator
Microsoft Multi-Factor Authentication (MFA) adds an extra layer of digital security for university accounts to access services such as Student Connect, Learning Managemnet System (LMS) and Student Connect. It requires both a password and a verification code to sign in in order to prevent unauthorised access.

## 4) SafeZone App

The SafeZone App allows students and staff to quickly alert security in an emergency and share their location. It also provides safety notifications and a check-in timer for monitoring personal wellbeing.
