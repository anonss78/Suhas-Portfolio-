## A8. Discover cryptography used in Internet of Things devices.

### Advanced Encryption Standard (AES)

A highly trusted encryption algorithm that protects sensitive data through direct format conversion. It utilises various key lengths (128,256 bits) to encrypt data and provides strong secuirty against unauthorised access. An widely known example is how it is implemented in internet communication where it encrypts shared messages and files.[1]
