# A14. Discover 5 AI-enabled security solutions.

## 1. Darktrace
 Uses machine learning to model normal network behaviour and instantly responds to deviations in real time.

## 2. CrowdStrike Falcon
A platform that detects malware on devices. It does this by analysing behavioural patterns rather than relying on signatures.

## 3. Microsoft Defender
Used machine learning to analyse to detect threats found in websites endpoints and personal emails.

## 4. Cylance (BlackBerry)
Utilises machine learning to predict and block malware before execution on the device, without needing frequent signature updates.

## 5. Vectra AI
 Monitors network traffic using AI to detect attacker behaviours like data exfiltration in real time.
