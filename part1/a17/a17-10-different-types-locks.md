***10 Different Types Of locks In Use***

1) Biometric Locks (Fingerprint/Eye Detection/Facial Recognition)

Uses an individual unique biological characteristics (fingerprint,iris and facial features) to authenticate an users identity in order to obtain permitted access. An example is the biometric facial recognition, ('Face ID') used on IPhones.


2) Smart Locks (Bluetooth/Wi-Fi):

 Keyless, connected locks that allows remote access, tracking user information and access times. A common example is the use of hotel keycards on vacation.

3) Electronic Keypad/PIN Locks: 

Requires a PIN or number code to unlock, commonly used on electronic lockers on the UWA campus in Reid Library to protect valuables.

4) RFID/NFC Keycard Locks: 

Using radio-frequency identification or near-field communication cards for access control, such as when using an elevator or unlocking a door.

5) Laptop Cable Locks: 

Physical cables, such as Kensington locks, that secure portable laptops to fixed objects, protecting physical hardware from theft.

6) Bike Lock:

Uses a chain to secure a bike frame to a stationary object (such as a bike rack) in order to prevent theft. 

7) Seat Belt Lock:

A technical device that prevents the seatbelt from extending. Used by police in order to restrain of detainees. 

8) Deadbolt 

A strong solid metal bolt that is turned into the door frame using a key, and requires the key to be unlocked. 

9) Padlock:

A portable lock with a shackle that can be opened using a key or combination. Commonly used on gates and storage units for basic security.

10) Combination Lock:

A lock that requires a sequence of numbers or symbols to open, rather than a key. Often used on school lockers, safes, and luggage.
