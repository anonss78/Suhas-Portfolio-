# A13. Discover 5 unique online security tools

## 5 Unique Online Security Tools:

### 1. VirusTotal:

VirusTotal scans multiple addresses for the detection of malware and other breaches using several antivirus engines.Users can upload files and receive a detailed report on security risks found and share them with community. 

### 2. Have I Been Pwned:

Have I Been Pwned allows individuals to check if their email addresses has been involved in data breaches and quickly take action to secure their compromised accounts.

### 3. SSL Lab's SSL Test

SSL Lab's SSL Test, assesses the security configuration of a website’s SSL/TLS certificates and protocols. It identifies weak ciphers and missing HSTS in order to improve web security.

### 4. Shodan

Shodan is a search engine for crawling internet-connected devices and services.It exposes vulnerabilities found in the connected servers and Internet Of Things devices.

### 5. OWASP ZAP (Zed Attack Proxy)

OWASP ZAP is a security scanner used for finding vulnerabilities in websites by simulating attacks like SQL injection and Cross Site Scripting (XSS) which improves the security of applications in the developement process.
